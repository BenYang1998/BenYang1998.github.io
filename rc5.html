MY FIRST WEEK IS OVER!

September 25, 2020 Ben Yang

This week has been really fun and productive in terms of just learning about Computer Science! Even though RC has been remote, I think I have been learning alot. I would have loved to do RC in person, so maybe I'll swing by for a second batch in the future. RC kind of feels like school with extracurriculars and study groups - the social aspect is a little absent, but I haven't gone to the music consumption groups after 5 PM, so I may join those and lurk in the future.

Anyways, here's what I did on Friday. I haven't done alot.

  • Finished three MongoDB / Phaser tutorials with slight minor bugs. Aiming to merge the three together in one project.
  • Read Chapters 2 & 3 of the Stanford online NLP book.
  • Finished the set of OverTheWire Bandit wargames. I do admit I googled a lot to finish them!

For next week:

  • Fix up the content security policy bug with the MongoDB chatroom tutorial and merge it with the card game code.
  • Think about how I'm going to set up my SNN AI App. (This task has been on here for a while.)
  • Finish the Coursera Policy Reinforcement course.
  • Do Leviathan and Krypton OverTheWire CTF challenges.
  • Make two shaders for this week's GPU Session!
  • Do Cryptopals Set 2.
  • Do Nand2Tetris Set 2.
  • Aim to do at least two leetcode sessions next week.

Here's my finalized list of everything that I plan on doing. Now, it's just time to execute.

  • Computer Security (Cryptopals, Pwnable.kr, Overthewire, Exploit.education)
  • Learn Haskell (by doing the CIS course and LearnHaskell Github)
  • Code Karuta
  • Code Tellstones
  • Mess around with SNNs and AI to make a Worldbuilding App
  • Read the NLP Textbook from Stanford
  • Learn about Shaders & Photorealistic Rendering & GPU Programming (Make a Minecraft Shader)
  • Build a computer through Nand2Tetris
  • Do leetcodes twice a week (excluding Week 1)
  • Finish the RL Coursera course & OpenAI Spinning Up Tutorial

I've also kept up to date on my new gym schedule (MWF instead of once every three days) and started following Tom Merrick (is that the correct spelling?) for flexibility training. So far, so good!
